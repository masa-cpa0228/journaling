# 思考ジャーナル（仮タイトル）

公認会計士・高橋正哉の **個人格コラム集** を作るための作業リポジトリです。

公式サイト [cpa-tm.com](https://www.cpa-tm.com/home) とは役割を明確に分け、
こちらは **「個人としての思考・読書・人生論を発信する場」** として作っていきます。

> 本リポジトリは GitHub・Web開発（vibecoding）の学習を兼ねており、
> 1ステップずつ小さく作っていく運用方針を採っています。

---

## このサイトの位置づけ

| | 公式サイト | 本サイト（思考ジャーナル） |
|---|---|---|
| 主体 | 法人格・専門家 | 個人格 |
| 役割 | サービス紹介・問い合わせ獲得 | 思考・コラムの発信 |
| 文体 | フォーマル | 読み物・エッセイ調 |
| トーン | 紺×金の重厚デザイン | 静かでシンプルな読み物デザイン |

---

## ローカルでの確認方法

このフォルダで以下を実行：

```bash
python3 server.py
```

ブラウザで `http://localhost:3000` を開くとサイトが表示されます。
（`server.py` は Python 標準機能だけで動く簡易Webサーバーです。）

## ファイル構成（最終形のイメージ）

```
.
├── index.html              トップページ
├── about.html              サイトについて
├── articles.html           コラム一覧
├── articles/               個別記事ページを格納
│   └── 2026-05-05-accountants-and-generative-ai.html
├── assets/
│   ├── css/main.css        共通スタイル
│   └── js/main.js          共通スクリプト（フェーズ2以降）
├── data/
│   └── articles.json       記事メタデータ（フェーズ2以降）
├── server.py               ローカル確認用Webサーバー
├── README.md               このファイル
└── PROGRESS.md             開発進捗の記録
```

> 上記は最終形です。最初は `index.html` と `server.py` だけがあり、
> ステップごとに少しずつファイルが増えていきます。

## 開発の進め方

- 詳しい開発計画とステップ分割は [PROGRESS.md](PROGRESS.md) を参照。
- **1ステップ ＝ 1ブランチ ＝ 1プルリクエスト** の単位で進めます。
- 各ステップ完了時に [PROGRESS.md](PROGRESS.md) を更新します。
- 「次に書きたい記事ネタ」「気になる改善点」は GitHub の Issue で別管理します。
